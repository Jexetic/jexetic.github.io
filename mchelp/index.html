Minecraft Server Help

November 28, 2022

This note contains some useful information and commands for Jexetic’s Minecraft server. Most of this functionality comes from Vanilla Tweaks datapacks.

Teleport to Spawn#

To teleport to spawn/our main area, type /trigger spawn 

Homes#

Each player can set up to 3 homes.

To set a home, type /trigger sethome set 1 (or use a different number to set a different home).

To teleport to a home, type /trigger home set 1  (or the number for whatever home you want to go to).

To delete a home, type /trigger delhome set 1  (or the number for the home you’d like to delete).

Teleporting to Players#

To teleport to a player, first hold tab and see what the yellow number to the right of their name is. 

Then, type /trigger tpa set 4  where 4  is the number you found next to their name.

To accept a teleport request, type /trigger tpaccept

Find your Grave#

To find your last death point, type /trigger grave 

Coordinates Hub#

You can display your current coordinates above your hotbar. Use the command /trigger ch_toggle  to turn this display on and off.
